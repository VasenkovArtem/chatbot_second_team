ChatBot websocket docs

Подключение к WebSocket:

ws://*наш_будущий_хост*/chat

Пример сообщения:

1;0;1 (формат - string)
[где первая первое значение - id сообщения,
второе значение - инентификатор сообщения(если нажата одна из предложенных кнопок - 0, если отправлено текстовое сообщение - 1)
третье значение - значение сообщения(если нажата кнопка - id кнопки, если отправлено сообщение - его значение]

Пример ответа:

{"id": "2", "text": "\u0417\u0430\u0440\u0435\u0433\u0438\u0441\u0442\u0440\u0438\u0440\u043e\u0432\u0430\u043d\u044b \u043b\u0438 \u0432\u044b \u0432 \u043a\u0430\u0447\u0435\u0441\u0442\u0432\u0435 \u0441\u0430\u043c\u043e\u0437\u0430\u043d\u044f\u0442\u043e\u0433\u043e?", "files": null, "is_buttons": true, "buttons": [{"name": 1, "text": "\u0414\u0430"}, {"name": 2, "text": "\u041d\u0435\u0442"}]}
[где текст представлен кириллицей в кодировке ISO 8859-5
files - ссылка на файл для скачивание при его наличии
"is_butons" - true, если отображаем кнопки
"buttons" - массив кнопок]
